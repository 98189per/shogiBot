# shogiBot
Discord bot for playing shogi made with Discord.js and PIL
![Shogi Piece](oushou.jpeg)
## Basic Commands
`s.help`  
`s.rules`  
`s.play`  
`s.accept`  
`s.style`  
`s.move`  
`s.drop`  
`s.undo`
